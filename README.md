OpenTerminal
============

A SublimeText plugin to add a **Open in Terminal** menu option...

### How to install

	# replace ~/.config/sublime-text-3/Packages/ with if necessary
	cd ~/.config/sublime-text-3/Packages/
	git clone https://github.com/fffilo/OpenTerminal.git
